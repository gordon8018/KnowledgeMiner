## 🚀 快速开始

### 最简使用方式

```bash
# 1. 进入项目目录
cd knowledge_compiler

# 2. 运行编译器
python -m src.main_cli --source ./你的文档目录 --output ./输出目录
```

就这么简单！系统会自动：
- 分析你的 Markdown 文档
- 提取关键概念
- 生成索引和分类
- 创建结构化知识库

## 🎯 基本使用

### 命令行模式

#### 基础用法

```bash
# 最简单的用法
python -m src.main_cli --source ./docs --output ./knowledge_base
```

#### 常用选项

```bash
# 启用交互模式（逐个确认提取的概念）
python -m src.main_cli --source ./docs --output ./knowledge_base --interactive

# 启用 LLM 增强（需要 API key）
python -m src.main_cli --source ./docs --output ./knowledge_base \
    --llm-provider openai \
    --model gpt-4

# 启用语义嵌入
python -m src.main_cli --source ./docs --output ./knowledge_base \
    --enable-embeddings

# 详细输出模式
python -m src.main_cli --source ./docs --output ./knowledge_base --verbose

# 静默模式（只显示错误）
python -m src.main_cli --source ./docs --output ./knowledge_base --quiet

# 使用配置文件
python -m src.main_cli --source ./docs --output ./knowledge_base \
    --config my_config.json
```

#### 实用组合

```bash
# 推荐配置：交互模式 + LLM + 详细输出
python -m src.main_cli \
    --source ./docs \
    --output ./knowledge_base \
    --interactive \
    --llm-provider anthropic \
    --model claude-3-sonnet-20240229 \
    --enable-embeddings \
    --verbose

# 批处理模式：非交互 + 静默
python -m src.main_cli \
    --source ./docs \
    --output ./knowledge_base \
    --no-interactive \
    --quiet
```

### 3. 语义嵌入

#### 启用嵌入生成

```python
from src.core.config import get_config

config = get_config()

# 启用嵌入
config.embeddings.enabled = True
config.embeddings.model = "text-embedding-ada-002"
config.embeddings.batch_size = 100

compiler = KnowledgeCompiler(config)
results = compiler.compile()

# 访问嵌入向量
concepts = compiler.extracted_concepts
for concept in concepts:
    if concept.embedding is not None:
        print(f"{concept.name}: 嵌入维度 {len(concept.embedding)}")
```

#### 语义搜索

```python
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

# 假设我们有一些带有嵌入的概念
concepts_with_embeddings = [c for c in concepts if c.embedding is not None]

# 查询概念
query_concept = concepts_with_embeddings[0]
query_embedding = query_concept.embedding.reshape(1, -1)

# 计算相似度
similarities = []
for concept in concepts_with_embeddings:
    embedding = concept.embedding.reshape(1, -1)
    similarity = cosine_similarity(query_embedding, embedding)[0][0]
    similarities.append((concept.name, similarity))

# 排序并显示最相似的
similarities.sort(key=lambda x: x[1], reverse=True)
print(f"与 '{query_concept.name}' 最相似的概念:")
for name, sim in similarities[:5]:
    print(f"  {name}: {sim:.2f}")
```

## 🧠 Phase 2 知识发现

Phase 2 提供了强大的知识发现引擎，可以自动发现概念之间的关系、模式和洞察。

### 基础知识发现

```python
from src.discovery import KnowledgeDiscoveryEngine, DiscoveryConfig
from src.main import KnowledgeCompiler
from src.core.config import get_config

# 1. 首先编译文档
config = get_config()
config.source_dir = "./docs"
config.output_dir = "./output"

compiler = KnowledgeCompiler(config)
compilation_result = compiler.compile()

# 2. 创建知识发现配置
discovery_config = DiscoveryConfig(
    enable_explicit_mining=True,    # 显式关系挖掘
    enable_implicit_mining=True,    # 隐式关系挖掘
    enable_statistical_mining=True, # 统计关系挖掘
    enable_semantic_mining=True,    # 语义关系挖掘
    enable_temporal_detection=True, # 时间模式检测
    enable_causal_detection=True,   # 因果关系检测
    enable_evolutionary_detection=True,  # 演化模式检测
    enable_conflict_detection=True  # 冲突检测
)

# 3. 创建发现引擎
discovery_engine = KnowledgeDiscoveryEngine(discovery_config)

# 4. 运行知识发现
discovery_result = discovery_engine.discover(
    documents=compiler.processed_documents,
    concepts=compiler.extracted_concepts,
    relations=compiler.extracted_relations  # 可选：已有关系
)

# 5. 查看发现结果
print(f"发现的关系: {len(discovery_result.relations)}")
print(f"检测的模式: {len(discovery_result.patterns)}")
print(f"发现的缺口: {len(discovery_result.gaps)}")
print(f"生成的洞察: {len(discovery_result.insights)}")
```

### 交互式知识探索

```python
from src.discovery import InteractiveDiscovery

# 创建交互式探索器
interactive = InteractiveDiscovery(discovery_engine)

# 存储发现结果
interactive.discover_and_store(
    documents=compiler.processed_documents,
    concepts=compiler.extracted_concepts,
    relations=discovery_result.relations
)

# 探索特定概念的关系
momentum_relations = interactive.explore_relations("动量指标")
print(f"动量指标的关系:")
for relation in momentum_relations:
    print(f"  {relation.source} -> {relation.target}: {relation.relation_type}")

# 查找特定模式
trading_patterns = interactive.find_patterns("交易策略")
print(f"\n交易策略模式:")
for pattern in trading_patterns:
    print(f"  {pattern.pattern_type}: {pattern.description}")

# 分析领域缺口
technical_gaps = interactive.analyze_gaps_in_domain("技术分析")
print(f"\n技术分析领域的缺口:")
for gap in technical_gaps:
    print(f"  {gap.gap_type}: {gap.description}")

# 获取顶级洞察
top_insights = interactive.get_top_insights(10)
print(f"\n最重要的洞察:")
for insight in top_insights:
    print(f"  [{insight.significance:.2f}] {insight.title}")
    print(f"    {insight.description}")
```

## ❓ 常见问题

### Q1: 如何选择 LLM 提供商？

**A:** 根据需求选择：

- **OpenAI GPT-4**: 最佳质量，成本较高
- **Anthropic Claude**: 平衡质量和成本
- **Ollama (本地)**: 免费，隐私好，质量稍低
- **不使用 LLM**: 最快，但功能受限

### Q2: 如何提高概念提取准确度？

**A:** 调整这些参数：

```python
config.processing.min_confidence_threshold = 0.8  # 提高阈值
config.processing.max_concepts_per_document = 10  # 减少数量
config.llm.temperature = 0.3  # 降低温度，更精确
```

### Q3: 如何处理大型文档集合？

**A:** 使用增量编译和批处理：

```python
# 启用状态管理
config.state.enabled = True
config.state.file = "./state.json"

# 批处理
config.embeddings.batch_size = 200

# 分批处理
# 第一次运行
compiler.compile()

# 后续只处理修改的文件
```
